# gh-star-repos-cli
> :octocat: Get a Github user repos by star

## Install

```
$ npm install --global gh-star-repos-cli
```

## Usage

```
$ gh-star-repos --help

  Usage
    gh-star-repos [input]

  Options
    --foo  Lorem ipsum [Default: false]

  Examples
    $ gh-star-repos
    unicorns & rainbows
    $ gh-star-repos ponies
    ponies & rainbows
```
